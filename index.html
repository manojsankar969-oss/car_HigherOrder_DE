Numerical Integration Method
Algorithm: Runge-Kutta 4th Order (RK4)
This method numerically solves the differential equation by breaking time into small steps and calculating forces at each step. RK4 is highly accurate for engineering applications.

Process:
  • Calculate spring force: F_s = -kx
  • Calculate damping force: F_d = -cv
  • Calculate external force: F_ext = F(t)
  • Total acceleration: a = (F_ext + F_d + F_s) / m
  • Update velocity and position for next time step
  • Repeat 1000 times to build complete motion profile
